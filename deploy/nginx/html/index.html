Workflow

How it works

A simple three-step loop that adds human judgment to any autonomous agent.

1

Agent Sends a Request

Your agent calls Verifi with an intent and a claim. A free entitlement or the 0.10 USDC entry payment admits it to the human queue.

2

Human Reviews & Responds

A human responder reviews the context and makes the judgment call. Requests that violate policy are rejected instead of answered.

3

Agent Receives the Result

Your agent polls until the result is ready, then unlocks it through the separate 2.90 USDC gate. Free chains perform the same unlock for $0.

Quickstart

Submit, poll, unlock

Send an intent and a claim. A human reviews it and returns a verdict. 

# Your wallet address receives 5 complete free chains.
curl -X POST https://verifi.cloud/verify \
  -H "Content-Type: application/json" \
  -d '{
  "intent": "send_email",
  "claim": "The user explicitly opted in and confirmed their
  email address via double opt-in on 2026-07-15.",
  "agent_id": "0xYourWalletAddress"
}'

# Every admitted call returns HTTP 202 with a durable id.
{
  "verify_id": "a1b2c3d4-...",
  "status": "processing",
  "next_action": "poll",
  "retry_after_seconds": 15
}
# Poll GET /verify/{verify_id}, for minutes if needed.
# When status is ready, POST /verify-unlock?id={verify_id}.
# Paid: 0.10 USDC to enter, then a new 2.90 USDC unlock.
# Free: both explicit gates cost 0.00 USDC.
# Docs: verifi.cloud/docs
