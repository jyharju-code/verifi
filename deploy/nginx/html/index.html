When an AI agent needs judgment, it can ask a human.

Verifi gives agents a durable way to buy a real human decision. Pay 0.10 USDC to enter the queue, poll for as long as the work takes, then unlock the ready answer for 2.90 USDC.

Fair Trade Intelligence

The human is not hidden inside the API. Human judgment is the product.

AI agents are fast and capable. They still encounter claims, decisions, language, and consequences that deserve a person.

Verifi turns that escalation into an explicit economic relationship. The agent knows when it paid, the person gets a clear task, and the system records the request, the two gates, free entitlements, settlements, credits, and the final outcome.

For agents

A small interface for work that can take real time.

No account portal and no API key. The requester wallet is the identity, x402 is the payment layer, and the verify ID is the durable handle.

Durable from the first responseEvery admitted request returns a verify ID that survives disconnects, restarts, and long human response times.
Machine-readable next actionsPoll while processing, unlock when ready, stop on failed, and consume the answer on completed.
One active chain per walletThe agent cannot accidentally create a pile of parallel requests or pay twice for admission.
quickstart.sh 
# The wallet receives five complete free chains.
curl -X POST https://verifi.cloud/verify \
  -H "Content-Type: application/json" \
  -d '{
    "intent": "Review this customer update",
    "claim": "We restored service and will compensate affected users.",
    "agent_id": "0xYourWalletAddress"
  }'

# Keep the returned verify_id. Poll for minutes if needed.
curl https://verifi.cloud/verify/{verify_id}

# When status is ready, perform the explicit second gate.
curl -X POST "https://verifi.cloud/verify-unlock?id={verify_id}"
processingHuman work is pending.Wait and poll the same verify ID.
